Step-by-Step Guide

How to Submit Your Term Project

Complete all steps before the deadline. Each step takes about 5 minutes.

1

Create a GitHub Account

Every team member needs a free GitHub account. If you already have one, skip to Step 2.

  1. Go to github.com/signup
  2. Choose a professional username (e.g., firstname-lastname or your student ID)
  3. Verify your email address
Sign Up for GitHub
2

Create the Team Repository

One team member (the repository owner) creates a new public repository using the required naming convention.

Repository Naming Rule
db2026-team<XX>
Replace <XX> with your two-digit team number assigned by the instructor.
Example: db2026-team03
Steps to create the repository:
  1. Click the + icon in GitHub's top-right corner → New repository
  2. Set Repository name to db2026-team<XX>
  3. Set visibility to Public ✔
  4. Check Add a README file
  5. Click Create repository
The repository must be Public so the instructor can verify your submission.
3

Add Team Members as Collaborators

Invite all teammates so everyone can push commits to the repository.

  1. Open your repository on GitHub
  2. Go to Settings → Collaborators
  3. Click Add people and search by each teammate's GitHub username or email
  4. Each teammate accepts the invitation from their email or GitHub notifications
Collaborators get Write access by default, which is exactly what you need.
4

Develop Your Project & Commit Regularly

Keep all project files in the repository. Commit often to show your progress.

Option A — GitHub Web Interface (no installation needed)
  1. Open the repository on github.com
  2. Click Add file → Upload files
  3. Drag and drop your files, write a commit message, click Commit changes
Option B — Git Command Line
# Clone the repo to your computer (run once)
git clone https://github.com/<owner>/db2026-team<XX>.git
cd db2026-team<XX>

# After making changes, commit and push
git add .
git commit -m "Add ER diagram and SQL schema"
git push
Option C — Visual Studio 2022 Recommended for this project

Visual Studio has built-in Git & GitHub integration — no separate Git installation needed.

  1. Sign in to GitHub inside VS
    Tools → Options → Accounts → Add → GitHub
  2. Clone your repository
    Git → Clone Repository → paste your GitHub URL → Clone
  3. Create your .NET / C# solution inside the cloned folder
    File → New → Project → save it into the cloned folder so VS tracks it automatically
  4. Commit your changes
    View → Git Changes → stage files → write a message → Commit All
  5. Push to GitHub
    Click the ↑ Push button in Git Changes, or Git → Push
Useful VS shortcuts
Ctrl+Shift+GOpen Git Changes
Ctrl+`Open Terminal
Git menu → Fetch / PullGet teammates' changes
Add a .gitignore
When creating the solution, check Add to source control and VS auto-generates a .gitignore that excludes large build folders (bin/, obj/). If not, add one at: github.com/github/gitignore
Tip: Meaningful commit messages like "Add normalization analysis for 3NF" or "Implement enrollment stored procedure" help the instructor see your progress and may earn participation credit.
5
